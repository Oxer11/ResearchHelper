# Research Helper

### Install

```bash
# Clone this repository
$ git clone git@github.com:Oxer11/ResearchHelper.git

# Install dependencies for back-end
$ pip3 install django django-cors-headers djangorestframework validators

# Install dependencies for front-end
$ npm install
```


### Usage

```bash
# Run back-end server
$ cd backend && python manage.py runserver 8001

# Launch Research Helper Website
$ npm start

# Start the site at the local host http://127.0.0.1:8000/
```
